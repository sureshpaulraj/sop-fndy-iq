## 1. Purpose

This SOP establishes the emergency evacuation procedures for all RCCB warehouse and office facilities. Every employee must be familiar with this plan and participate in quarterly evacuation drills.

## 2. Scope

Covers all emergency situations requiring facility evacuation:
- Fire or explosion
- Hazardous material release
- Severe weather (tornado warning)
- Structural damage
- Security threat
- Utility failure creating unsafe conditions

## 3. Alarm Systems

| Alarm Type | Signal | Meaning |
|-----------|--------|---------|
| Fire Alarm | Continuous horn + strobe | Evacuate immediately |
| Hazmat Alarm | Intermittent horn (3 blasts) | Evacuate — chemical release |
| Tornado Warning | Steady siren (60 seconds) | Shelter in place |
| All Clear | Single long tone (10 seconds) | Safe to return |

## 4. Evacuation Procedure

### 4.1 When the Alarm Sounds
1. **STOP** all work immediately
2. Secure hazardous operations if safe to do so (max 30 seconds):
   - Park forklifts and turn off ignition
   - Close chemical containers
   - Shut down powered equipment
3. **DO NOT** collect personal belongings
4. **DO NOT** use elevators

### 4.2 Exit the Building
1. Proceed to the **nearest marked exit** — follow illuminated EXIT signs
2. Walk — do NOT run
3. Close doors behind you (do NOT lock)
4. Assist persons with disabilities using buddy system
5. If smoke is present: stay low, cover mouth with cloth

### 4.3 Assemble at Muster Point

| Zone | Muster Point | Assembly Leader |
|------|-------------|----------------|
| Zone A (Warehouse North) | North Parking Lot — Flag Post A | Warehouse Supervisor |
| Zone B (Warehouse South) | East Loading Dock Staging Area | Shipping Supervisor |
| Zone C (Office Building) | South Visitor Parking Lot | Office Manager |
| Zone D (Maintenance) | West Equipment Yard | Maintenance Lead |

### 4.4 Accountability
1. Department leads/supervisors conduct **headcount** at muster point
2. Use the Emergency Roster (posted at each muster point)
3. Report status to Incident Commander within **10 minutes**:
   - ✅ All accounted for
   - ⚠️ Missing persons — provide names and last known location
   - 🚑 Injured persons — provide names and injury description

## 5. Special Situations

### 5.1 Visitors and Contractors
- Hosts are responsible for their visitors during evacuation
- Contractors report to their designated site contact
- All visitors must sign in at reception (provides accountability data)

### 5.2 Disabled or Injured Persons
- Each person with a mobility limitation has an assigned evacuation buddy
- Evacuation chairs are located at each stairwell landing
- If unable to evacuate: proceed to Area of Rescue (stairwell landing), call 911

### 5.3 After-Hours Evacuation
- Security monitors alarm systems 24/7
- On-call supervisor is notified automatically
- Night shift employees follow same evacuation routes
- Security conducts sweep of facility

## 6. Incident Commander Responsibilities

The **Incident Commander** (IC) is the senior manager on-site:
1. Establish command post at the designated location (near main entrance)
2. Coordinate with emergency services (fire dept, EMS, police)
3. Receive headcount reports from assembly leaders
4. Authorize search and rescue only by trained personnel
5. Issue **ALL CLEAR** only after:
   - Emergency services confirm the building is safe
   - All personnel are accounted for
   - Hazards have been mitigated

## 7. Emergency Equipment Locations

| Equipment | Locations |
|-----------|-----------|
| Fire extinguishers | Every 75 feet along travel paths |
| AED (defibrillator) | Main entrance, break room, shipping office, each muster point |
| First aid kits | Break room, receiving office, shipping office, each muster point |
| Evacuation chairs | Stairwell landings (2nd floor office) |
| Emergency eyewash | Chemical storage, battery charging, maintenance |
| Emergency phone | Each dock door, main entrance, break room |

## 8. Drill Schedule and Requirements

- **Frequency:** Quarterly (minimum 4 per year)
- **Duration target:** Full evacuation in under 5 minutes
- **Documentation:** Record drill date, evacuation time, issues found
- **Corrective actions:** Address any issues within 30 days
- **Annual review:** Update plan with any facility changes
